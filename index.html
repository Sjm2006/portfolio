The system intelligently generates structured schedules, tracks progress through analytics like focus score and streaks, and visualizes performance with charts — all while storing data locally in the browser.

ATMOS BY SJM

Atmos is a responsive weather application that delivers live meteorological data for any searched location. Users can enter a city name to fetch current conditions like temperature, humidity, wind speed, atmospheric pressure, and an extended 5-day forecast, helping them plan their day or week with up-to-date weather intelligence.

JavaScript HTML5 CSS3 Weather API Real-Time Data
